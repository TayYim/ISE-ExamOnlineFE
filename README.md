# ISE-ExamOnlineFE

在线答题系统-前端

> A Vue.js project
> Using vuex,vue-router,iView,axios

***baseUrl need to be modified before connected to the backend***

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
